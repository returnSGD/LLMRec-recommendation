# 当前论文缺少的关键内容（对比 Nature 期刊标准）

## 1. 结构性缺项（Nature 期刊强制要求）

| 缺失项 | 说明 |
|--------|------|
| Data Availability 声明 | 每篇 Nature 论文都有独立的数据可获取性声明，说明数据来源、获取方式、是否有访问限制 |
| Code Availability 声明 | 所有 Nature 论文明确标注代码发布地址（GitHub/Zenodo）及 DOI |
| Ethics Declaration | 伦理审批、知情同意、出版许可，Nature 期刊必须 |
| Competing Interests | 所有作者的利益冲突声明 |
| Author Contributions | 每位作者的具体贡献（CRediT 分类法，如：概念化、方法、实验、写作） |
| Acknowledgments / Funding | 基金号、资助机构、致谢 |
| Author Affiliations + ORCID | 作者单位、通讯作者邮箱、ORCID |
| Supplementary Information 索引 | 明确列出所有补充材料及其组织方式 |

## 2. 统计严谨性 ✅ P0 已完成 (2026-05-26)

当前论文已有完整统计检验。详见 `results/statistical_significance_report.md`。

- ✅ 所有关键比较需要显著性检验：已实现配对 t 检验 + Wilcoxon signed-rank test + FDR 多重比较校正
- ✅ 需要报告标准差/置信区间：论文中所有 LLM 方法结果已更新为 mean ± SD 格式，含 Bootstrap 95% CI
- ✅ 需要多次重复实验：已支持多 seed 训练框架（`trainer.py --seed X`），seed=456 的 5K×3ep 实验正在运行中

## 3. 实验深度

**缺失的分析: 组件消融（Component Ablation）**  
Nature 论文参考: Rocket 文消融了 stage size、policy model size、scheduler 类型等；Tactiformer 消融了 role encoding、zone encoding、team attention 等每个模块

**缺失的分析: 超参数敏感性分析**  
Nature 论文参考: Rocket 详细分析了 learning rate、momentum、batch size 等 6+ 个超参数的变化趋势和影响

**缺失的分析: 定性案例分析**  
Nature 论文参考: 大多数 Nature 论文都有具体的 case study / qualitative examples

**缺失的分析: 鲁棒性/扰动分析**  
Nature 论文参考: ChartRecover 在各种扰动条件下测试；Tactiformer 做了 occlusion masking 测试

**缺失的分析: 计算代价分析**  
Nature 论文参考: Rocket 报告了训练时间、加速比、成本节省；当前论文只简单提到了“训练太慢”

**缺失的分析: 跨数据集泛化**  
Nature 论文参考: Rocket 在 8 个任务 32 个 benchmark 上验证；当前论文只有 1 个数据集

## 4. 图表质量

- 无误差棒（error bars）：当前论文所有表格只有均值，无标准差/标准误
- 无统计显著性标注：无 * p<0.05, ** p<0.01 等标记
- 无可视化轨迹图：Rocket 用 t-SNE 可视化超参数搜索轨迹，用折线图展示超参数变化趋势
- 无模型架构图：缺少整体的框架图来说明 RecCL/SANS/RecAug 三者之间的关系

## 5. 论文组织

- 缺少 Plain Language Summary：Scientific Reports 要求为非专业读者写通俗解释（Tactiformer 文有“Plain Explanation”章节和“Analogy or Intuition”术语表）
- 缺少结构化方法章节：Nature 论文的 Methods 通常独立成章，包含详细的公式推导、实现细节、评估协议
- 讨论不够深入：Nature 论文的 Discussion 通常 3-5 页，包含：主要发现→与已有工作的关系→意外发现→局限性→未来方向→更广泛影响

## 6. 开放性

- 无 Source Data 文件（Nature 要求所有图表背后有可下载的源数据）
- 无预注册/预印本链接
- 无 Reporting Summary（Nature 的标准化实验报告清单）
